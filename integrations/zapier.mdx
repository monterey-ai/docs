## Set up

Create a Zapier account and go to the [integration settings page](https://app.monterey.ai/w/default/settings/integrations/) to set it up. You will need to use an [API key](https://app.monterey.ai/w/default/settings/api/) to do so.
To get started, watch the instructional video for setup examples. You can combine Zapier triggers with other app triggers or with Zapier's default triggers to create workflows and automations.

## Access

Only admins of your Monterey workspace can set up Zapier integration.
