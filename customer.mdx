## Usage

Monterey will group all users who provided feedback together on the "Customer" page. You can find the user's name, email, user ID, and all of their feedback in one place. To locate a specific customer, you can use the search filter by name. 


## Segments

You can create customized customer segments and assign attributes to them in order to better organize your customers.You can add multiple segament tags to one customer. 
The default segments are "Enterprise", "SMB" and "Individual" to help you get started. You can add or edit them in the [user segments page](../https://app.monterey.ai/w/cl5b6pdc45011ebfde64e4qkf/settings/user-segments).
