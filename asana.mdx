## Set up
Go to the [intigration setting page](../https://app.monterey.ai/w/cl5b6pdc45011ebfde64e4qkf/settings/integrations/) and select the Asana integration option. Follow the guidelines to connet your Asana account. Only admins of your Monterey workspace can set up Asana integration.
