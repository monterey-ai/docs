## Set up

Create a Zapier account and find the [intigration setting page](../https://app.monterey.ai/w/cl5b6pdc45011ebfde64e4qkf/settings/integrations/) to set it up. You will need to use an [API key](../https://app.monterey.ai/w/clfwzomuy0002l91nmtfatmea/settings/api/) to do so. To get started, watch the instructional video for setup examples. You can combine Zapier triggers with other app triggers or with Zapier's default triggers to create workflows and automations.

## Access

Only admins of your Monterey workspace can set up Zapier integration.
