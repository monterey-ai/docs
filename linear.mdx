## Set up

Go to the [integration settings page](https://insights.reforge.com/w/default/settings/integrations/) page and select the Linear integration. Follow the guidelines to connect your Linear account. Only admins of your Reforge Insight Analytics workspace can set up Linear integration.

## Usage

You can create a ticket for any individual feedback from the Reforge Insight Analytics workspace. Click on each feedback and you will see a feedback detail view with an action bar to the right.
At the bottom of the action bar, click the "Generate issue" button which will bring up a form requiring a title, team and assignee. You can also optionally include a description and project.
After clicking on "Create issue", the Linear ticket will be created and found in the specified issue list or you can find it in My issues \> created in Linear.

![linear](https://monterey-documentation-gifs.s3.us-east-2.amazonaws.com/linear.gif)

## User Access

Anyone who has access to your workspace can create a ticket in Linear, even if they do not have access to Linear themselves. Only users of Linear will be able to view and manage the ticket in Linear.
