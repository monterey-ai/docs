## Install Chrome extension

Install the Chrome extension from the [store](https://chrome.google.com/webstore/detail/monterey-ai/faoofmdmlogpijhcobkhilajiphlbnpl).

## Usage

Monterey's Chrome Extension allows your team to tag customer feedback right from any app with a single highlight and click.
