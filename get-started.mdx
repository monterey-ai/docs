## Get access

Monterey is currently in limited Beta. The quickest way to get access to Monterey is to sign up on monterey.ai/pricing, and we will get back to you as soon as possible with an invite.


## Platform

Monterey is a web application and supports browsers like Chrome and Safari. You can also access Monterey in mobile but the experience is not optimized there yet.
