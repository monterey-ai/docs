## Get access

Monterey is currently in limited Beta. The quickest way to get access to Monterey is to sign up on [Monterey.ai](https://www.monterey.ai/). We will get back to you as soon as possible with an invitation.


## Platform

Monterey is a web application that supports browsers like Chrome and Safari. You can also access Monterey in mobile but the experience is not yet optimized.
